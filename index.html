Lediga jobb Se jobbsektionen nedan

Real Logic är ett konsultföretag som hjälper våra kunder öka tillväxten i ett snabbt föränderligt samhälle.

Snabb förändring och utveckling i samhället kräver nytt tänkande och nya tillvägagångssätt för att öka tillväxten. Från idé till produkt, varumärke till upplevelse, kunder till verksamhet, vi ger den insikt, noggrannhet och expertis som krävs för att både lyfta fram och förverkliga dessa möjligheter.

Karriär

Utveckla dig själv och din karriär

Vi brinner lika mycket för att utveckla våra medarbetare som för att hjälpa våra kunder växa. Vi motiverar våra anställda att tänka utanför boxen, driva idéer och se möjligheter. Från junior till erfarna bidrar vi alla till det tänkande som våra kunder förväntar sig.

Expertisområden

Real Logics verksamhet är baserad på unik expertis inom informations- och mjukvaruområdena samt dess tillämpningar.

Denna kombination har resulterat i en utpräglad förmåga att optimera kunders utveckling och produktivitet.

Våra tjänster

Våra tjänster baseras på Java-plattformen, JavaScript och tillhörande ramverk samt övriga webbtekniker.

Områden

Webbutveckling, APIutveckling, Apputveckling, Cloud, Machine learning, AI, IoT, Automation och QA

Roller

Vi agerar i roller som Utvecklare, Tech lead, DevOps, IT-arkitekter och Lösningsarkitekter.

Arbeta med oss som partner eller underleverantör

Real Logic är alltid intresserade av att prata med dig som är riktigt duktig inom ditt område. Vi är öppna att samarbeta med underleverantörer eller registrera dig som partner för att öka värdet av den service vi tillhandahåller våra kunder. Vi använder ofta underleverantörernas tjänster för att hjälpa oss med ett visst projekt för en viss tid för att vi ska kunna fortsätta att leverera de höga krav som våra kunder väntar sig.

Vi söker företag eller enskilda personer som är kvalificerad att arbeta inom områden såsom Webbutveckling, APIutveckling, Apputveckling, Cloud, Machine learning, AI, IoT, Automation och QA i roller såsom Utvecklare, Tech lead, DevOps, IT-arkitekter och Lösningsarkitekter.

Kontakta oss för att diskutera hur vi kan hjälpa varandra!
